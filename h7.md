# h7 - SecuDream

## a) SecuDream(tm). Cyber security course ... of my dreams.

### Goals for the course

1. Able to outline key cybersecurity practices, concepts, frameworks etc. and identify how they are implemented in the students own organization / how they could be improved.

2. Able to outline valid and up-to-date sources for cybersecurity knowledge, news etc. and how to identify good sources from bad ones

3. Hands on experiences on basic tools and methods related to cybersecurity (e.g. virtualization, password cracking,  port scanning etc.)

4. Get the student personally invested and interested in Cyber Security (e.g. has to have some sort of motivation to pick the course -> cultivate and inspire that motivation)
